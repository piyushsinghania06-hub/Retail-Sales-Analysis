# Retail Sales Analysis & Prediction

## Overview
This project performs end-to-end analysis on a retail sales dataset to uncover trends, customer behavior, and sales predictions.

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sales Trend Visualization
- Top Product Analysis
- Machine Learning Prediction Model

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
1. Load Dataset
2. Clean Data
3. Perform EDA
4. Visualize Insights
5. Train Prediction Model
6. Evaluate Results

## Dataset
Retail sales dataset containing:
- Order Date
- Product Category
- Sales
- Profit
- Quantity

## Results
- Identified top-performing categories
- Detected seasonal sales trends
- Predicted future sales with regression model
